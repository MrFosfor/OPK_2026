# Универсальный кодер/декодер

Программа позволяет шифровать и дешифровать текст с использованием четырех алгоритмов.

Поддерживается два языка: Русский и Английский.

Доступные шифры:
1. **Цезарь** - Сдвиг букв на заданное число 
2. **Atbash** - Зеркальная замена букв в алфавите ("А" -> "Я", "Я" -> "А")
3. **Хеш-подстановка** - Динамический сдвиг (Сдвиг зависит от позиции символа в строке)
4. **Квадрат Плейфера** - Шифрование биграм по матрице (без "J" в англ версии и без "Ё" в русской)
