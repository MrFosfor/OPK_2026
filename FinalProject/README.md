# Универсальный кодер/декодер

Программа позволяет шифровать и дешифровать текст с использованием четырех алгоритмов.

Поддерживается два языка: Русский и Английский.

Доступные шифры:
1. **Цезарь** - сдвиг букв на заданное число
2. **Atbash** - зеркальная замена букв в алфавите
3. **Хеш-подстановка** - динамический сдвиг
4. **Квадрат Плейфера** - шифрование биграм по матрице
